Image Rotator

I think the title should be pretty self-explanatory ... this is an example of a webpage/application with a rotating image.

I chose the image of the Sun as the image to be rotated for two main reasons:

  1. The Sun (as depicted in this cartoon image) has a lot of lines of rotational symmetry; therefore it is a perfect candidate for image rotation.
  2. The only reason I wanted to test this out was because of this webpage: http://summer.ust.hk/cgi-bin/2015/index.php

Thank you, W3Schools, for teaching me how to create a Web Worker which enabled me to create this animation.
